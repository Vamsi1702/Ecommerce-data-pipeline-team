# E-Commerce Data Pipeline Project

#  Gold Layer – KPIs & Analytics

##  Overview
This notebook represents the **Gold layer** of the data pipeline, where cleaned data is transformed into **business-ready insights (KPIs)**.

---

##  Objective
To convert processed data from the Silver layer into meaningful metrics that support business decision-making.

---

##  KPIs Implemented
- Daily Revenue  
- Monthly Revenue  
- Top Customers  
- Top Products  
- Category Performance  
- Average Delivery Time  

---

##  Approach
- Used the **sales_detail (fact table)** from the Silver layer  
- Applied aggregations using **PySpark**  
- Generated KPI tables and stored them in the **Gold schema**  

---

##  Tools Used
- PySpark  
- Delta Lake  
- Databricks  

---

##  Outcome
The Gold layer provides **optimized, aggregated data** that is directly used for dashboards and business analysis.
